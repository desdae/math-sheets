Visual Companion

Math worksheet builder for families, teachers, and students.

This board turns the product brief into a concrete v1 shape: a printable worksheet generator, saved solving sessions, personal stats, and competitive leaderboards backed by PostgreSQL and secured with Google OAuth.

Vue 3 + Vite frontend Node + Express API PostgreSQL relational schema Google Sign-In Print-first worksheet UX

Primary User Flow

1. Discover and sign in Landing page explains the product, demo worksheet visuals, and Google login.
2. Generate worksheet User picks count, difficulty, operations, number ranges, and worksheet size.
3. Solve and save progress Answers are editable, auto-saved manually or explicitly saved as draft/partial.
4. Submit for scoring Backend evaluates correctness, stores per-question results, updates stats.
5. Compare and review Dashboard shows history, streak-like activity, and daily/weekly/monthly leaderboards.

Recommended v1 Architecture

Client App

Vue Router, Pinia, composables for auth/API state, print-friendly worksheet templates.

API Layer

Express routes with auth middleware, validation, worksheet generation service, stats queries.

Data Layer

Normalized PostgreSQL tables with aggregate-friendly indexes and leaderboard SQL views.

Recommendation Use JWT access tokens with secure HTTP-only refresh-token cookies. This keeps the frontend simple, supports Google OAuth cleanly, and avoids storing long-lived bearer tokens in local storage.

Relational

Analytics
